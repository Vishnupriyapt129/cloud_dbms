This project is a Cloud File Storage and Access Control System developed using Python and MySQL.
It provides a secure platform for users to upload, manage, and share files over the internet. 
The system includes user authentication with encrypted passwords and implements role-based access control (RBAC) to ensure secure file sharing and permission management. 
It allows users to perform file upload and download operations, organize files through categorization, and manage access rights efficiently. 
An admin dashboard is also included for monitoring system activities. 
The project demonstrates effective integration of a Python-based application with a MySQL database, focusing on secure data handling, user management and database-driven authorization.
